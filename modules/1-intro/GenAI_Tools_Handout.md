# GenAI Tools Reference Guide

Throughout this course please reference this GenAI Tools Guide which is organized by generation modality. This document will be updated regularly, however, the world of AI advances quicker than we can update these resources. Some items may be outdated. This course does not require you to pay for a subscription to any AI model, in fact the free tier of many GenAI platforms are amazingly powerful. The free tiers will likely come with limitations such as daily/monthly rate limits, but we've structured this course to ideally fit within those limitations. You may find it beneficial to use a few different free platforms and in fact we strongly encourage that approach.

---

## 1. Text Generation (LLMs)

| Tool | Best For | Free Tier | Paid Tier |
|------|----------|-----------|-----------|
| **ChatGPT (OpenAI)** <https://chatgpt.com> | Writing, reasoning, multimodal chat | • Lightweight model • Shorter context window • Slower responses • Limited file/image analysis • Basic reasoning and coding | • Frontier models • Long context window • Faster responses + higher rate limits • Reliable file/image/audio analysis • Strong reasoning & coding performance |
| **Claude (Anthropic)** <https://claude.ai> | Careful reasoning, long documents | • Smaller model • Short-to-medium context • Limited uploads • Moderate reasoning reliability | • High-accuracy model • Very long context window • Large uploads (PDFs, docs) • Strong analytical & structured writing abilities |
| **Gemini (Google)** <https://gemini.google.com> | Web-integrated reasoning, multimodal | • Lightweight model • Basic multimodal features • Basic browsing-style results • Slower, less accurate reasoning | • Advanced model • Long context & faster performance • Strong multimodal understanding • Better factual accuracy & reasoning |
| **Perplexity** <https://perplexity.ai> | Research-style answers w/ citations | • Free research mode • Limited depth of citations • Uses smaller models • Basic web access | • Access to premium models (GPT, Claude) • Deeper synthesis + better citations • Faster & more consistent outputs |
| **Copilot (Microsoft)** <https://copilot.microsoft.com> | Coding + Office tools | • Basic chat + lightweight model • Limited coding suggestions • No Office integration | • Advanced reasoning model • Integrated into Word/Excel/PowerPoint • Strong coding + debugging support |
| **LMArena** <https://lmarena.ai> | Free, limited comparison between models | Basic interactions with many different models | N/A |
| **NotebookLM** <https://notebooklm.google.com> | RAG-based chat, audio generation, video, etc. | 50 sources. Limited generations | More sources, faster generation. (Free tier is sufficient for many applications) |

**Notes:**
- Free tiers = smaller models, shorter memories, limited uploads, slower speeds.
- Paid tiers = better reasoning, longer documents, richer multimodal abilities, and higher reliability.
- These patterns apply across all major tools (ChatGPT, Claude, Gemini, Perplexity, Copilot).

## 8. Other Helpful Resources

| Resource | Notes |
|----------|-------|
| **Futuretools.io** <https://futuretools.io> | A large, regularly updated directory of AI tools across many categories (text, image, audio, productivity, research, etc.). Helpful for discovering new tools, comparing capabilities, and staying current with the quickly changing GenAI ecosystem. |
| **LMArena** <https://lmarena.ai> | A public leaderboard and "battle arena" where you can compare top AI models side-by-side without knowing which model you're testing. Great for observing differences in reasoning, tone, accuracy, and creativity across the most capable frontier and open-source models. "Battle mode" is default — select "Direct Chat" to choose a model. |
| **Hugging Face** <https://huggingface.co> / <https://huggingface.co/chat> | A central hub for open-source AI models, datasets, and demos. You can try models directly in the browser, explore how they're trained, download them for local use, or examine model cards to understand limitations, capabilities, and licensing. Excellent for transparency and experimentation. |
